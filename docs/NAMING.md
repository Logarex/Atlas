# Naming

## Product Name

The working product name is Atlas.

Use Atlas as the app display name, README title, local package name, and GitHub repository name if the target namespace is available.

## Availability Check

Checked on 2026-05-16:

- `https://github.com/louischabert/atlas` returned HTTP 404, so the repo name appears available under that GitHub namespace from a public unauthenticated check.
- `https://github.com/atlas` returned HTTP 200, so `atlas` is not globally unused on GitHub.
- Public search results show existing App Store and Google Play apps using Atlas in their names.

## Practical Decision

Atlas is a good working name, but it is generic and crowded. Before public launch:

- Reserve `louischabert/atlas` or the chosen GitHub organization repo.
- Check App Store Connect name availability in the target countries.
- Check Google Play Console name availability.
- Run a proper trademark search in the launch markets if the app becomes popular.

If the exact app name is blocked, keep the repository as `atlas` and publish the store listing as a clearer variant such as `Atlas: Apple Store Guide` or `Atlas Retail`.
