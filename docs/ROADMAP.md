# Roadmap

## Phase 0: Foundation

- Choose stack and project name.
- Publish README, privacy policy, contribution guide, data policy.
- Create mobile app shell.
- Create data schema and first sourced sample record.
- Create Supabase schema.

## Phase 1: App MVP

- Store list and detail pages.
- Map screen with markers.
- English/French language switching from device locale.
- Supabase connection.
- Username profiles.
- Mark store as visited with date, including local-only use without an account.
- Basic search and filters.

## Phase 2: Data and Moderation

- In-app correction form.
- New store proposal form.
- Pending photo uploads.
- Discord webhook notifications.
- Reviewer dashboard.
- Accepted changes audit trail.
- Data validation in CI.

## Phase 3: Community

- Friend requests.
- Public/private profile settings.
- Shared visited-store lists.
- Contributor credits.
- Trust levels for frequent contributors.

## Phase 4: Sharing and Polish

- Generated visit cards.
- Instagram Stories/share sheet integration.
- Offline cache.
- Advanced map filters.
- Accessibility pass.
- App Store and Play Store launch preparation.

## Phase 5: Complete Atlas

- Worldwide open stores.
- Permanently closed stores.
- Relocated stores and previous addresses.
- Architectural eras and attributes.
- Public API and dataset exports.
