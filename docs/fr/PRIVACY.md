# Politique de Confidentialité

Atlas est un projet open-source, dirigé par la communauté, conçu par et pour les passionnés de l'Apple Store. Parce qu'il ne s'agit pas d'une entreprise, nous ne nous soucions pas de vos données. L'application est conçue pour être aussi privée que techniquement possible.

## Vos données restent sur votre appareil (Local First)

Atlas ne possède pas de comptes utilisateurs, de bases de données ou de serveurs de suivi. 
Tout ce que vous faites dans l'application est enregistré exclusivement sur votre propre téléphone :
- Vos visites en boutique
- Vos notes personnelles
- Vos photos privées
- Vos préférences d'application

Personne, y compris le développeur, n'a accès à ces données. Si vous supprimez l'application, vos données sont supprimées avec elle. Si vous souhaitez les sauvegarder, vous pouvez utiliser la fonction manuelle "Exporter" dans les paramètres de l'application pour enregistrer un fichier sur votre appareil.

## Analyses et Suivi

Nous n'utilisons aucun logiciel d'analyse, rapporteur de plantage ou SDK de suivi. Nous ne savons pas comment vous utilisez l'application, quelles boutiques vous visitez ou où vous vivez.

## Contributions Publiques

La seule fois où les données quittent votre appareil, c'est si vous choisissez explicitement de contribuer au projet Atlas. 

Si vous utilisez les formulaires intégrés pour **Suggérer une modification**, **Proposer une boutique manquante** ou **Soumettre une photo publique** :
- Les informations que vous fournissez (nom, photo et détails de la boutique) seront envoyées à notre dépôt GitHub public.
- Parce qu'elles sont envoyées à GitHub, ces informations deviennent publiques et peuvent être consultées par tout le monde.
- Nous ne collectons pas votre e-mail ni aucun identifiant d'appareil caché pendant ce processus.

## Modifications

Si nous modifions un jour la façon dont l'application gère les données, nous mettrons à jour ce document. Mais comme la philosophie fondamentale d'Atlas est de rester un projet open-source et local d'abord (Local First), votre vie privée sera toujours la priorité.

## Contact

Si vous avez des questions concernant la confidentialité, veuillez ouvrir une "issue" sur notre [dépôt GitHub](https://github.com/Logarex/Atlas).
