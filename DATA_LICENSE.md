# Data License Policy

This project separates code, factual metadata, photos, and third-party map data.

## Code

Application code is licensed under the MIT License in `LICENSE`.

## Curated Factual Metadata

Atlas aims to publish curated factual store metadata under CC0 1.0 where possible. This includes community-entered facts such as store names, coordinates, opening dates, closure dates, architectural tags, and source links.

Every accepted contribution to curated metadata must include a source and must be contributed under CC0-compatible terms.

## Photos

User-uploaded photos are not CC0 by default. Contributors must select an explicit license at upload time. The recommended default is Creative Commons Attribution 4.0 International (CC BY 4.0), because it allows reuse while preserving credit.

Do not upload Apple-owned photos, press images, screenshots, marketing imagery, or images copied from other apps unless the license clearly permits redistribution.

## Third-party Data

OpenStreetMap data is licensed under ODbL. If this project ever imports OSM-derived data into the canonical dataset, those derived records must be isolated and distributed with the required ODbL terms and attribution.

Map tiles, geocoding providers, and external datasets may have separate licenses. Their data must not be mixed into the CC0 canonical dataset unless their terms permit it.
